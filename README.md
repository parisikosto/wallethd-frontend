# wallethd-frontend

> Frontend application for WalletHD — a modern income and expense tracking tool built with React and TypeScript

WalletHD’s frontend delivers a clean, intuitive, and responsive interface for managing your personal finances. Built with React and TypeScript, it provides seamless access to your income and expense data, detailed insights, and visual analytics — empowering you to take full control of your financial life from any device.

## Getting Started

Start by cloning this repository

```
git clone https://github.com/parisikosto/wallethd-frontend.git
```

Install dependencies

```
make install
```

Start local development

```
make start
```
